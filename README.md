# Miro Web Studio – oma portfolio

Tämä on moderni portfolio-sivusto, joka on tehty laadukkaalla rakenteella HTML:llä, CSS:llä ja JavaScriptillä.

## Rakenne

```text
index.html
assets/
  favicon.svg
styles/
  base.css
  layout.css
  components.css
  responsive.css
scripts/
  data.js
  ui.js
  main.js
README.md
```

## Mitä sivulla on?

- Responsiivinen desktop- ja mobiiliversio
- Hero-osio
- Projektikortit: Kahvila Lumi, NordClean ja Vihreä Piha
- Palvelut-osio
- Miksi valita minut -osio
- Hinnat alkaen -osio
- Yhteydenottolomake
- Tarjouspyyntöikkuna
- Projektien lisätietoikkunat
- Mobiilivalikko
- Scroll-animaatiot

## Julkaisu GitHub Pagesilla

1. Avaa repositoryn **Settings**
2. Mene kohtaan **Pages**
3. Valitse **Deploy from a branch**
4. Branch: **main**
5. Folder: **/(root)**
6. Paina **Save**

Sivun osoite on yleensä:

```text
https://miromnb-coder.github.io/oma-portfolio/
```

## Huomio

Lomakkeet näyttävät onnistumisviestin demona. Oikeassa käytössä ne voidaan yhdistää esimerkiksi sähköpostipalveluun, Formspreehen tai muuhun lomaketyökaluun.
